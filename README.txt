Tutorial penginstallan aplikasi:
1. Pertama-tama, download NodeJS dulu di halaman websitenya https://nodejs.org/
2. Download NodeJS sesuai OS yang anda pakai, pilih NodeJS versi terbaru/stable
3. Setelah download NodeJS selesai, pastikan anda melakukan instalasi full NodeJS nya dengan file installer yang sudah di download tadi
4. Buka terminal/CMD kemudian ketikan "node -v" (tanpa kutip), jika NodeJS berhasil diinstal maka akan menampilkan versi NodeJS yang anda install
5. Setelah NodeJS berhasil diinstall dengan benar, masuk ke direktori anda menyimpan file aplikasi. Misal C:/Downloads/TuruAuto
6. Masuk dengan command cd <path/to/directory>
7. Setelah itu, jalankan lagi command npm install
8. Setelah menginstall dependency secara total, jalankan aplikasi dengan command node main
9. Jika cara yang dilakukan sudah benar, aplikasi akan langsung melakukan automattic visit ke url yang anda inginkan
10. Selesai
